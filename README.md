## Description

RESTful API for management of users and their posts.

## Technologies

Nodejs, Nest.js framework, Postgresql, prisma, passport, pactumJS, docker etc...

## Installation

```bash
$ npm install
```

## Running the app

```bash

# watch dev mode

$ npm run db:dev:restart

$ npm run start:dev

```

## Test

I wrote some e2e tests to test all REST API.

```bash

# e2e tests

$ npm run pretest:e2e

$ npm run test:e2e

```
